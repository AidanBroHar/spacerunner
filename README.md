# SpaceRunner

SpaceRunner: Compile any ML model for radiation-hardened space hardware. Write in any framework. Deploy to orbit.

# Motivation

I believe elegant solutions to hard problems drive progress. Satellites enable Earth observation, disaster response, and autonomous systems that benefit humanity. Removing friction from satellite ML deployment lets more people build on this potential.
